# Weather app

This is just me toying around with the free NOAA api. 
There is nothing objectively helpful that this application provides other than me having a bit of fun. 

I chose not to use venv as I didn't feel like it. Will probably implement that later. 

```
pip install requests
pip install flask
```

Run using the following command in the same directory
```
Flask --app weather run
```
Then navigate to `http://127.0.0.1:5000`

*this does launch the developer application as I have given this a whole 1 day of effort*
